Apa itu NPM

NPM (Node Package Manager) adalah manajer paket untuk Node.js yang digunakan untuk mengelola dependensi paket/library dalam proyek JavaScript. NPM adalah alat yang memudahkan kita untuk menginstal, memperbarui, dan menghapus paket yang diperlukan dalam proyek kita. Dengan NPM, kita dapat dengan mudah mengelola berbagai library dan alat yang digunakan dalam pengembangan aplikasi web, seperti React, Express, Tailwind CSS, dan banyak lagi.



Fungsi Utama NPM

  • Install package/library dari internet
  • Menyimpan informasi dependensi di file package.json
  • Mengelola versi dari library tersebut
  • Mem-publish (upload) package buatan sendiri ke internet (npm registry)



Perintah Dasar NPM

  • npm init: Membuat file package.json baru.
  • npm install [package-name]: Menginstal package/library dari npm registry.
  • npm uninstall [package-name]: Menghapus package/library dari proyek.
  • npm update: Memperbarui semua package/library ke versi terbaru.
  • npm list: Menampilkan daftar semua package/library yang terinstal dalam proyek.

index.html
package.json
node_modules/
server.js

4. Apa fungsi dari folder node_modules/?

Tempat menyimpan file gambar
Tempat menyimpan file HTML
Tempat menyimpan file konfigurasi
Tempat menyimpan semua package/library yang diinstal melalui NPM

5. Apa perbedaan antara npm install dan npm install [nama_package]?

npm install untuk satu package, yang satu untuk semua
npm install menginstal semua dependensi dari package.json, sedangkan npm install [nama_package] menginstal satu package tertentu
Tidak ada perbedaan
npm install untuk menghapus package
