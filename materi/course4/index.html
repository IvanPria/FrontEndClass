JavaScript Dasar

Dasar-Dasar JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif. Dengan JavaScript, kita dapat menangani event, memanipulasi DOM, dan membuat animasi di dalam browser.


Fungsi JavaScript:

  • Menambahkan interaktivitas pada halaman web seperti pop-up, animasi, dan efek transisi.
  • Memanipulasi elemen HTML secara dinamis tanpa perlu memuat ulang halaman.
  • Mengontrol perilaku form dan validasi input pengguna sebelum dikirim ke server.
  • Berkomunikasi dengan server melalui AJAX untuk pengambilan data secara real-time.
  • Mendukung pengembangan aplikasi web modern dengan framework seperti React, Vue, dan Angular.

Contoh Struktur Dasar JavaScript

                        <script>
                            // Menampilkan pesan ke dalam console
                            console.log("Hello, JavaScript!");
                        </script>

Variabel dalam JavaScript

Variabel digunakan untuk menyimpan data dalam JavaScript. Ada tiga kata kunci utama untuk mendeklarasikan variabel:

  • var - Memiliki cakupan fungsi dan dapat diubah.
  • let - Memiliki cakupan blok dan lebih aman digunakan.
  • const - Memiliki cakupan blok dan nilainya tidak dapat diubah setelah dideklarasikan.
                        let nama = "John";
                        const umur = 25;
                        console.log(nama + " berumur " + umur + " tahun.");

Fungsi dalam JavaScript

Fungsi adalah blok kode yang dapat digunakan kembali untuk menjalankan tugas tertentu.

                        function sapa(nama) {
                            return "Halo, " + nama + "!";
                        }
                        console.log(sapa("Alice"));

Event Handling

JavaScript memungkinkan kita untuk menangani berbagai event seperti klik, hover, dan input pengguna.

                        <button onclick="alert('Tombol diklik!')">Klik Saya!</button>

Fungsi dan Scope dalam JavaScript

Fungsi dalam JavaScript adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi membantu dalam modularisasi kode dan meningkatkan keterbacaan serta pemeliharaan kode.


1. Definisi Fungsi

Fungsi dapat dideklarasikan dengan menggunakan kata kunci function atau menggunakan arrow function (=>).

                        
                // Deklarasi fungsi
                function sapa(nama) {
                    return "Halo, " + nama + "!";
                }
                
                // Arrow function
                const sapaArrow = (nama) => "Halo, " + nama + "!";
                
                // Pemanggilan fungsi
                console.log(sapa("Andi"));
                console.log(sapaArrow("Budi"));

2. Parameter dan Return

Fungsi bisa menerima parameter dan mengembalikan nilai dengan menggunakan return.

                        
                function tambah(a, b) {
                    return a + b;
                }
                
                console.log(tambah(5, 3)); // Output: 8

3. Scope dalam JavaScript

Scope menentukan di mana variabel dapat diakses dalam kode. Terdapat tiga jenis scope:

  • Global Scope - Variabel yang dideklarasikan di luar fungsi bisa diakses di mana saja.
  • Local Scope = Variabel yang dideklarasikan di dalam fungsi hanya bisa diakses dalam fungsi tersebut.
  • Block Scope - Variabel yang dideklarasikan dengan let atau const hanya bisa diakses dalam blok tempat ia dideklarasikan.
                        
                let globalVar = "Saya global";
                
                function contohScope() {
                    let localVar = "Saya lokal";
                    console.log(globalVar); // Bisa diakses
                    console.log(localVar); // Bisa diakses
                }
                
                console.log(globalVar); // Bisa diakses
                // console.log(localVar); // Error! localVar tidak bisa diakses di luar fungsi

4. Hoisting dalam Fungsi

JavaScript secara otomatis mengangkat (hoist) deklarasi fungsi ke atas, sehingga kita bisa memanggil fungsi sebelum dideklarasikan.

                        
                console.log(katakanHalo("Andi")); // Bisa dipanggil sebelum deklarasi
                
                function katakanHalo(nama) {
                    return "Halo, " + nama + "!";
                }

DOM (Document Object Model) Manipulation dalam JavaScript

DOM (Document Object Model) adalah representasi struktur halaman web dalam bentuk pohon yang memungkinkan JavaScript untuk memanipulasi elemen HTML secara dinamis.

1. Mengakses Elemen

JavaScript menyediakan berbagai metode untuk mengakses elemen dalam DOM.

                            // Mengakses elemen berdasarkan ID
                            var heading = document.getElementById("judul");
                            
                            // Mengakses elemen berdasarkan kelas
                            var items = document.getElementsByClassName("item");
                            
                            // Mengakses elemen berdasarkan tag
                            var paragraphs = document.getElementsByTagName("p");
                            
                            // Menggunakan querySelector
                            var firstItem = document.querySelector(".item");

2. Mengubah Konten dan Atribut

DOM memungkinkan kita untuk mengubah teks dan atribut dari elemen HTML.

                            // Mengubah teks elemen
                            document.getElementById("judul").innerText = "Judul Baru";
                            
                            // Mengubah HTML elemen
                            document.getElementById("content").innerHTML = "Konten Baru";
                            
                            // Mengubah atribut elemen
                            document.getElementById("link").setAttribute("href", "https://example.com");

3. Menambahkan dan Menghapus Elemen

Kita dapat menambahkan atau menghapus elemen dari halaman web menggunakan JavaScript.

                            // Membuat elemen baru
                        var newParagraph = document.createElement("p");
                        newParagraph.innerText = "Ini paragraf baru";
                        document.body.appendChild(newParagraph);
                        
                        // Menghapus elemen
                        var elementToRemove = document.getElementById("hapus");
                        elementToRemove.remove();

4. Menangani Event

Event memungkinkan kita untuk merespon aksi pengguna seperti klik, hover, atau input.

                            // Menambahkan event listener untuk tombol
                            document.getElementById("btn").addEventListener("click", function() {
                                alert("Tombol diklik!");
                            });

JavaScript Asynchronous Programming

Asynchronous Programming dalam JavaScript memungkinkan kode untuk berjalan tanpa harus menunggu operasi lain selesai. Hal ini sangat berguna untuk menangani operasi seperti permintaan jaringan, animasi, atau operasi berjangka panjang.

1. Callback Functions

Callback adalah fungsi yang dikirim sebagai argumen ke fungsi lain dan dieksekusi setelah tugas selesai.

                            function prosesData(callback) {
                                setTimeout(() => {
                                    console.log("Proses selesai");
                                    callback();
                                }, 2000);
                            }
                    
                            prosesData(() => {
                                console.log("Lanjut ke proses berikutnya");
                            });

2. Promises

Promise digunakan untuk menangani operasi asynchronous dengan lebih rapi dibandingkan callback.

                            let janji = new Promise((resolve, reject) => {
                                setTimeout(() => {
                                    let sukses = true;
                                    if (sukses) {
                                    resolve("Data berhasil didapatkan");
                                } else {
                                    reject("Terjadi kesalahan");
                                }
                            }, 2000);
                        });
                        
                        janji.then(response => console.log(response))
                        .catch(error => console.log(error));

3. Async/Await

Async/Await adalah cara modern untuk menangani operasi asynchronous dengan sintaks yang lebih mudah dibaca.

                            async function fetchData() {
                                try {
                                    let response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
                                    let data = await response.json();
                                console.log(data);
                            } catch (error) {
                                console.log("Terjadi kesalahan: ", error);
                            }
                        }
                        
                        fetchData();

ES6+ (Fitur Modern JavaScript)

ES6 (ECMAScript 2015) dan versi selanjutnya menghadirkan berbagai fitur modern yang membuat JavaScript lebih efisien dan mudah dibaca. Berikut beberapa fitur utama ES6+:

1. Let dan Const

let dan const digunakan untuk mendeklarasikan variabel dengan cakupan blok.

                            let nama = "John";
                            const umur = 25;
                            console.log(nama, umur);

2. Arrow Function

Arrow function adalah cara singkat menulis fungsi anonim.

                            const sapa = (nama) => console.log(`Halo, ${nama}!`);
                            sapa("Alice");

3. Template Literals

Gunakan backticks (`) untuk membuat string lebih mudah dibaca dan disisipkan variabel.

                            let nama = "Bob";
                            let pesan = `Selamat datang, ${nama}!`;
                        console.log(pesan);

4. Destructuring

Destructuring memungkinkan ekstraksi nilai dari array atau objek dengan mudah.

                            let orang = { nama: "Charlie", usia: 30 };
                        let { nama, usia } = orang;
                        console.log(nama, usia);

5. Spread dan Rest Operator

Operator spread (...) digunakan untuk menyalin elemen array atau properti objek.

                            let angka = [1, 2, 3];
                            let angkaBaru = [...angka, 4, 5];
                            console.log(angkaBaru);

Fetch API

Fetch API adalah antarmuka JavaScript modern untuk membuat permintaan HTTP di peramban web. Fetch menyediakan cara yang lebih canggih dan fleksibel untuk mengirim dan menerima data dibandingkan dengan metode lama seperti XMLHttpRequest. Fetch menggunakan Promises, yang memungkinkan kode asinkron yang lebih bersih. Fetch mendukung berbagai format data, header kustom, dan berbagai jenis permintaan (GET, POST, dll.). API ini dirancang agar dapat diperluas dan terintegrasi dengan baik dengan teknologi web lainnya. Meskipun lebih sederhana untuk kasus penggunaan dasar, Fetch juga menangani skenario kompleks seperti pembatalan permintaan dan membaca respons yang dialirkan. Fetch didukung secara luas di peramban modern dan telah menjadi standar untuk permintaan jaringan dalam aplikasi JavaScript sisi klien.
