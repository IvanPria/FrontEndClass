HTML Dasar

Apa itu HTML

HTML adalah singkatan dari HyperText Markup Language. Bahasa ini digunakan di frontend dan memberikan struktur pada halaman web yang dapat Anda tata gayanya menggunakan CSS dan dibuat interaktif menggunakan JavaScript.

HTML menggunakan elemen-elemen seperti <head>, <body>, <h1>, <p>, <a>, <img>, dan lainnya untuk mengatur teks, gambar, tautan, dan elemen lainnya dalam sebuah halaman web.


Struktur Dasar HTML

HTML (HyperText Markup Language) memiliki struktur dasar yang harus diikuti agar halaman web dapat ditampilkan dengan benar di browser. Struktur ini terdiri dari beberapa elemen utama yang membentuk kerangka dasar sebuah halaman web.


1. Elemen Utama HTML


  • <!DOCTYPE html> → Menentukan bahwa dokumen ini menggunakan HTML5.
  • <html> → Elemen utama yang menampung seluruh isi halaman web.
  • <head> → Berisi informasi tentang halaman web seperti judul dan metadata.
  • <body> → Berisi konten utama yang ditampilkan di browser.

2. Contoh Struktur Dasar HTML


                        <!DOCTYPE html>
                        <html lang="id">
                            <head>
                                <meta charset="UTF-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <title>Halaman Pertama Saya</title>
                                <link rel="stylesheet" href="style.css">
                            </head>
                            <body>
                                <h1>Selamat Datang di Website Saya</h1>
                                <p>Ini adalah halaman web pertama saya menggunakan HTML.</p>
                            </body>
                        </html>

Elemen dan Tag HTML

Dalam HTML, elemen dan tag digunakan untuk menyusun dan menampilkan konten di halaman web.


1. Jenis-Jenis Elemen HTML

a. Elemen Blok

Elemen blok selalu memulai baris baru dan memenuhi lebar penuh.

                            <div>Ini elemen blok</div>
                            <p>Paragraf adalah elemen blok</p>
b. Elemen Inline

Elemen inline tidak memulai baris baru dan hanya selebar kontennya.

                            <span>Ini elemen inline</span>
                            <a href="#">Tautan adalah elemen inline</a>

2. Contoh Elemen HTML yang Sering Digunakan

  • <h1> hingga <h6> → Menampilkan judul.
  • <p> → Menampilkan paragraf teks.
  • <a> → Membuat tautan ke halaman lain.
  • <img> → Menampilkan gambar.
  • <ul> dan <ol> → Membuat daftar.
  • <table> → Membuat tabel.
  • <form> → Membuat formulir input.

Atribut pada HTML

Atribut digunakan untuk memberikan informasi tambahan pada elemen HTML. Format umum atribut: <elemen nama_atribut="nilai">Isi Elemen</elemen>


1. Contoh Atribut Global

Atribut yang dapat digunakan di hampir semua elemen.

  • <p> (paragraf) 
                                        <p id="paragraf1" class="teks-biru" style="font-size: 18px;" title="Ini adalah paragraf pertama">
                                    Ini adalah contoh atribut global.
                                    </p>

2. Contoh Atribut Khusus

Atribut yang dapat digunakan di hanya dibeberapa elemen.

  • <a> (Tautan) 
                                            <a href="https://www.google.com" target="_blank">Kunjungi Google</a>
  • <img> (Gambar) 
                                            <img src="gambar.jpg" alt="Gambar tidak ditemukan" width="200">
  • <input> (Formulir) 
                                        <input type="text" placeholder="Masukkan nama Anda">

Media dalam HTML

Media dalam HTML digunakan untuk menampilkan gambar, audio, video, dan iframe dalam halaman web.


1. Gambar (<img>)

Elemen <img> digunakan untuk menampilkan gambar.

<img src="gambar.jpg" alt="Gambar tidak ditemukan" width="200">

2. Audio (<audio>)

Elemen <audio> digunakan untuk menampilkan pemutar audio.

<audio controls>
                        <source src="audio.mp3" type="audio/mpeg">
                        Browser Anda tidak mendukung pemutar audio.
                        </audio>

3. Video (<video>)

Elemen <video> digunakan untuk menampilkan pemutar video.

<video width="300" controls>
                        <source src="video.mp4" type="video/mp4">
                        Browser Anda tidak mendukung pemutar video.
                        </video>

4. Iframe (<iframe>)

Elemen <iframe> digunakan untuk menyematkan halaman web lain dalam halaman kita.

<iframe src="https://www.example.com" width="300" height="200"></iframe>

Form dan Input pada HTML

Formulir dalam HTML digunakan untuk mengumpulkan data dari pengguna melalui berbagai jenis input.

1. Elemen Formulir (<form>)

Elemen <form> digunakan untuk membuat formulir.

<form action="submit.php" method="POST">...</form>

2. Input Teks (<input>)

Input teks memungkinkan pengguna memasukkan teks.

<input type="text" name="nama" placeholder="Masukkan nama Anda">

3. Input Email (<input>)

Input email memungkinkan pengguna memasukkan alamat email.

<input type="email" name="email" placeholder="Masukkan email Anda">

4. Input Password (<input>)

Input password menyembunyikan karakter yang diketik oleh pengguna.

<input type="password" name="password" placeholder="Masukkan kata sandi">

5. Tombol Kirim (<button>)

Tombol digunakan untuk mengirimkan data formulir.

<button type="submit">Kirim</button>
