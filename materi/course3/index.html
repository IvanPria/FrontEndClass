CSS Fundamental

Apa itu CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan desain elemen dalam halaman web. CSS memungkinkan pengembang untuk mengontrol warna, font, layout, animasi, dan berbagai aspek tampilan lainnya secara terpisah dari struktur HTML.


Fungsi CSS:

  • Mempercantik tampilan website - Mengatur warna, font, ukuran, dan jarak antar elemen.
  • Membantu responsivitas - Menyesuaikan tampilan agar cocok di berbagai perangkat (desktop, tablet, dan mobile).
  • Meningkatkan efisiensi kode - Dengan satu file CSS, banyak halaman HTML bisa memiliki gaya yang sama.
  • Memisahkan struktur dan desain - HTML untuk struktur, CSS untuk tampilan.

Selektor dan Spesifisitas CSS

Selektor CSS digunakan untuk memilih dan menata elemen HTML, sedangkan spesifisitas menentukan prioritas dari aturan CSS yang berlaku.


1. Jenis Selektor CSS

  • * → Universal (Memilih semua elemen)
  • element → Selektor Tag (Misal: p, h1)
  • .class → Selektor Kelas (Misal: .container)
  • #id → Selektor ID (Misal: #header)
  • selector1, selector2 → Selektor Grup
  • selector1 selector2 → Selektor Turunan

2. Contoh Penggunaan Selektor

                    /* Selektor tag */
                    p {
                        color: blue;
                    }
                    
                    /* Selektor kelas */
                    .highlight {
                        background-color: yellow;
                    }
                    
                    /* Selektor ID */
                    #header {
                        font-size: 24px;
                    }
                    
                    /* Selektor grup */
                    h1, h2, h3 {
                        font-family: Arial, sans-serif;
                    }
                    
                    /* Selektor turunan */
                    div p {
                        font-style: italic;
                    }

3. Spesifisitas CSS

Spesifisitas menentukan aturan mana yang lebih kuat dalam konflik aturan CSS. Urutan prioritasnya:

  • Inline Style (Paling kuat) → <p style="color: red;">
  • ID Selektor → #id
  • Kelas, Pseudo-Kelas, dan Atribut → .class, :hover
  • Elemen dan Pseudo-Element → h1, ::before

Properti dan Nilai CSS

CSS menggunakan berbagai properti untuk mengatur tampilan elemen HTML. Setiap properti memiliki nilai tertentu yang menentukan bagaimana elemen tersebut ditampilkan.


1. Properti Warna

  • color → Mengatur warna teks.
  • background-color → Mengatur warna latar belakang elemen.
                    /* Contoh penggunaan */
                    p {
                        color: blue;
                        background-color: lightgray;
                    }

2. Properti Teks

  • font-size → Mengatur ukuran teks.
  • text-align → Mengatur perataan teks (left, center, right).
  • line-height → Mengatur tinggi baris teks.
                    /* Contoh penggunaan */
                    h1 {
                        font-size: 24px;
                        text-align: center;
                        line-height: 1.5;
                    }

3. Properti Box Model

  • margin → Mengatur jarak luar elemen.
  • padding → Mengatur jarak dalam elemen.
  • border → Menambahkan garis tepi pada elemen.
                    /* Contoh penggunaan */
                    div {
                        margin: 20px;
                        padding: 15px;
                        border: 2px solid black;
                    }

4. Properti Tampilan dan Posisi

  • display → Mengatur bagaimana elemen ditampilkan (block, inline, flex, grid).
  • position → Mengatur posisi elemen (static, relative, absolute, fixed, sticky).
                    /* Contoh penggunaan */
                    .container {
                        display: flex;
                        position: relative;
                    }

5. Properti Flexbox

  • display: flex → Mengaktifkan model Flexbox.
  • justify-content → Mengatur perataan horizontal (center, space-between, space-around).
  • align-items → Mengatur perataan vertikal (flex-start, center, flex-end).
                    /* Contoh penggunaan */
                    .flex-container {
                        display: flex;
                        justify-content: center;
                        align-items: center;
                    }

Layout dan Posisi dalam CSS

CSS menyediakan berbagai properti untuk mengatur tata letak (layout) dan posisi elemen dalam halaman web.


1. Model Box

Model box terdiri dari margin, border, padding, dan content.

                    /* Contoh penggunaan */
                    .box {
                        width: 200px;
                        padding: 20px;
                        border: 2px solid black;
                        margin: 10px;
                    }

2. Display

Properti display menentukan bagaimana elemen ditampilkan di dalam dokumen.

  • block → Elemen tampil dalam baris penuh.
  • inline → Elemen tampil dalam satu baris.
  • flex → Menggunakan model Flexbox.
  • grid → Menggunakan model Grid.
                    /* Contoh penggunaan */
                    .container {
                        display: flex;
                    }

3. Posisi Elemen

Properti position mengontrol cara elemen diposisikan di dalam halaman.

  • static → Posisi default.
  • relative → Elemen diposisikan relatif terhadap posisi aslinya.
  • absolute → Elemen diposisikan relatif terhadap elemen induk yang memiliki posisi selain static.
  • fixed → Elemen tetap di tempat meskipun halaman di-scroll.
  • sticky → Elemen mengikuti scroll hingga mencapai batas tertentu.
                    /* Contoh penggunaan */
                    .fixed-box {
                        position: fixed;
                        bottom: 10px;
                        right: 10px;
                        background-color: red;
                        color: white;
                        padding: 10px;
                    }

4. Flexbox

Flexbox digunakan untuk mengatur layout secara fleksibel.

  • justify-content → Mengatur perataan elemen secara horizontal.
  • align-items → Mengatur perataan elemen secara vertikal.
                    /* Contoh penggunaan */
                    .flex-container {
                        display: flex;
                        justify-content: center;
                        align-items: center;
                    }

5. CSS Grid

Grid digunakan untuk mengatur layout berbasis kolom dan baris.

  • grid-template-columns → Menentukan jumlah kolom.
  • grid-template-rows → Menentukan jumlah baris.
  • gap → Mengatur jarak antara elemen grid.
                    /* Contoh penggunaan */
                    .grid-container {
                        display: grid;
                        grid-template-columns: 1fr 1fr;
                        gap: 10px;
                    }


Responsivitas dengan CSS

Responsivitas dalam CSS memungkinkan halaman web menyesuaikan tampilannya berdasarkan ukuran layar atau perangkat yang digunakan.


1. Media Queries

Media queries digunakan untuk menerapkan gaya CSS berdasarkan ukuran layar.

                    /* Contoh penggunaan */
                    @media (max-width: 600px) {
                        body {
                            background-color: lightblue;
                        }
                    }

2. Unit Relatif

Menggunakan unit relatif seperti %, em, rem, vw, vh untuk memastikan elemen berskala sesuai.

                    /* Contoh penggunaan */
                    .container {
                        width: 80%;
                        font-size: 1.2rem;
                    }

3. Flexbox untuk Layout Responsif

Flexbox membantu dalam menyusun elemen dengan fleksibel.

                    /* Contoh penggunaan */
                    .flex-container {
                        display: flex;
                        flex-wrap: wrap;
                        justify-content: space-between;
                    }

4. CSS Grid untuk Tata Letak Dinamis

Grid memungkinkan tata letak berbasis baris dan kolom yang lebih fleksibel.

                    /* Contoh penggunaan */
                    .grid-container {
                        display: grid;
                        grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
                        gap: 10px;
                    }

5. Gambar dan Video Responsif

Menggunakan properti max-width dan height: auto agar gambar dan video berskala otomatis.

                    /* Contoh penggunaan */
                    img, video {
                        max-width: 100%;
                        height: auto;
                    }


Animasi dan Transisi CSS

Animasi dan transisi dalam CSS memungkinkan elemen berubah secara halus dari satu gaya ke gaya lainnya.


1. Transisi CSS

Transisi memungkinkan perubahan gaya terjadi secara bertahap dalam rentang waktu tertentu.

                        /* Contoh penggunaan */
                        .btn {
                            background-color: blue;
                            color: white;
                            padding: 10px 20px;
                            transition: background-color 0.3s ease;
                        }
                        .btn:hover {
                            background-color: red;
                        }

2. Animasi CSS

Animasi memungkinkan elemen berubah secara bertahap menggunakan keyframes.

                        /* Contoh penggunaan */
                        @keyframes slideIn {
                            from {
                                transform: translateX(-100%);
                                opacity: 0;
                            }
                            to {
                                transform: translateX(0);
                                opacity: 1;
                            }
                        }
                    .box {
                        width: 100px;
                        height: 100px;
                        background-color: green;
                        animation: slideIn 1s ease-out;
                    }

3. Transformasi CSS

Transformasi memungkinkan elemen diputar, diperbesar, atau dipindahkan.

                        /* Contoh penggunaan */
                        .rotate-box {
                        width: 100px;
                        height: 100px;
                        background-color: orange;
                        transform: rotate(45deg);
                    }
