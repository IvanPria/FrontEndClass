Roadmap Belajar Frontend Development

2. HTML Dasar

Memahami selectors, box model, layout, flexbox, grid, dan responsive design.

⚡

4. JavaScript Dasar

Belajar tentang variabel, tipe data, fungsi, objek, array, DOM manipulation.

⚛️

5. Framework Frontend

Pengenalan ke React, Vue, atau Angular untuk pengembangan aplikasi modern.
