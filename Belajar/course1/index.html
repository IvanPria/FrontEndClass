Bagaimana Cara Kerja Internet

Internet adalah jaringan global komputer yang terhubung menggunakan protokol standar TCP/IP. Ketika Anda mengakses sebuah website, data berpindah melalui berbagai router dan server sebelum mencapai tujuannya.

Video: Penjelasan Cara Kerja Internet

Apa itu HTTP

HTTP (Hypertext Transfer Protocol) adalah protokol aplikasi untuk sistem informasi terdistribusi, kolaboratif, dan hypermedia. HTTP adalah fondasi komunikasi data untuk World Wide Web, di mana dokumen hypertext termasuk hyperlink ke sumber daya yang dapat diakses pengguna.

Video: Penjelasan HTTP dan HTTPS

Apa itu Domain Name

Domain name adalah nama yang mudah diingat yang ditetapkan ke alamat IP numerik. Alih-alih mengetik serangkaian angka (IP address), pengguna dapat mengetik nama domain seperti "google.com" untuk mengakses website.

Video: Penjelasan Domain Name

Apa itu Hosting

Hosting adalah layanan yang memungkinkan organisasi dan individu untuk menyimpan website atau aplikasi web di server dan membuatnya dapat diakses melalui internet. Server hosting menyimpan semua halaman, file, dan database yang membentuk website.

Video: Penjelasan Web Hosting

Apa itu DNS

DNS (Domain Name System) adalah sistem yang menerjemahkan nama domain ke alamat IP. Ketika Anda mengetik nama domain di browser, DNS server mencari alamat IP yang sesuai dan mengarahkan permintaan Anda ke server tersebut.

Video: Penjelasan DNS

Cara Kerja Browser

Browser web adalah aplikasi yang mengambil dan menampilkan halaman web. Browser mengirimkan permintaan ke server web, menerima respon (biasanya file HTML, CSS, dan JavaScript), dan menafsirkan file-file tersebut untuk menampilkan halaman web yang interaktif.

Video: Cara Kerja Browser
