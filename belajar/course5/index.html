Git & Github

Apa itu Git

Git adalah sistem kontrol versi terdistribusi yang dirancang untuk menangani proyek dengan ukuran apa pun dengan cepat dan efisien. Diciptakan oleh Linus Torvalds pada tahun 2005, Git melacak perubahan dalam kode sumber selama pengembangan perangkat lunak, yang memungkinkan banyak pengembang untuk bekerja sama dalam pengembangan non-linier. Git memberikan dukungan yang kuat untuk alur kerja pengembangan yang bercabang, menggabungkan, dan terdistribusi. Git menyimpan riwayat lengkap semua perubahan, yang memungkinkan pengembalian dan perbandingan yang mudah antar versi. Sifatnya yang terdistribusi berarti setiap pengembang memiliki salinan repositori yang lengkap, yang memungkinkan pekerjaan dan pencadangan offline. Kecepatan, fleksibilitas, dan kemampuan percabangan dan penggabungan Git yang kuat telah menjadikannya sistem kontrol versi yang paling banyak digunakan dalam pengembangan perangkat lunak, khususnya untuk proyek sumber terbuka.


Apa itu Github

GitHub telah menjadi pusat proyek sumber terbuka dan digunakan secara luas oleh pengembang, perusahaan, dan organisasi untuk repositori pribadi dan publik. GitHub diakuisisi oleh Microsoft pada tahun 2018 tetapi terus beroperasi sebagai entitas yang relatif independen. Popularitas GitHub telah menjadikannya alat penting dalam alur kerja pengembangan perangkat lunak modern dan platform utama untuk memamerkan proyek pengodean dan berkontribusi pada perangkat lunak sumber terbuka.

Latihan Soal

Soal 1 dari 5

1. Apa fungsi utama dari Git?

Menyimpan file secara online
Mengatur tampilan halaman web
Melacak perubahan dan versi dalam proyek
Mengedit gambar dalam proyek

2. Perintah Git apa yang digunakan untuk menyimpan perubahan yang sudah ditandai (staged)?

git push
git status
git add
git commit

3. Apa perbedaan utama antara Git dan GitHub?

Git untuk desain, GitHub untuk database
Git adalah sistem kontrol versi, GitHub adalah platform hosting online untuk proyek Git
Git untuk frontend, GitHub untuk backend
Git untuk membuat website, GitHub untuk mengelola konten

4. Perintah Git apa yang digunakan untuk menghubungkan repositori lokal ke repositori GitHub?

git push origin main
git init
git clone [url]
git remote add origin [url]

5. Apa yang dimaksud dengan commit dalam Git, dan apa fungsinya?

Menyimpan perubahan beserta pesan deskriptif ke dalam riwayat proyek
Mengunggah semua file ke internet
Internet System Protocol
Menghapus file dari repositori
